# Boilerplate Worker

**Last Updated:** April 2026

A template for creating new Cloudflare Workers in the Hoox ecosystem.

## Features

- Standard project architecture.
- Ready-to-use TypeScript configuration.
- Testing setup with `bun:test`.

## Setup

1. Copy this directory to `workers/your-worker-name`.
2. Update `package.json` and `wrangler.jsonc` with your worker's name.
3. Install dependencies:
   ```bash
   bun install
   ```

## Development

```bash
bun run dev
```

## Testing

```bash
bun run test
```
